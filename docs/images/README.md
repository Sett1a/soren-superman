# README Image Naming

Store README images in this directory and keep names tied to the UI content they show.

Current hero image:

- `workspace-overview-ai-editor-terminal.svg`

Recommended screenshot names for future README sections:

- `project-picker-and-recent-workspaces.png`
- `coding-cli-launcher-and-session-resume.png`
- `file-explorer-context-selection.png`
- `editor-selection-send-to-claude-code.png`
- `terminal-output-send-to-claude-code.png`
- `split-ai-and-terminal-panes.png`
- `git-branch-switcher-and-commit-graph.png`
- `repository-changes-review-panel.png`
- `repository-diff-workspace.png`

Naming rules:

- Use lowercase letters only.
- Separate words with hyphens.
- Name the file after the visible feature, not after a generic action like `screenshot`.
- Prefer `.png` for real screenshots and `.gif` only when motion is essential.
